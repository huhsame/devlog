## TL;DR
- Claude Code로 개발하면서 남긴 세션 로그를 자동으로 분석해서 개발노트로 정리하는 시스템을 만들었다
- 세션 로그 파싱 → 개발노트 작성 → Astro 블로그 배포까지 자동화
- 앞으로 여기에 바이브코딩 기록을 남길 예정

## 배경

Claude Code로 이것저것 만들다 보니, 매번 "오늘 뭐 했더라?" 하고 기억을 더듬게 됐다. 세션 로그에 다 남아있는데 정리를 안 하니까 날아가는 거다.

그래서 만들었다. Claude Code 세션 로그를 자동으로 수집하고, 개발노트 형식으로 정리해서, 이 블로그에 올리는 시스템.

## 구조

```
session-analyzer (세션 로그 수집/파싱)
       ↓
devlog-writer (개발노트 마크다운 생성)
       ↓
devlog-publisher (블로그 빌드/배포)
```

## 앞으로의 계획
- 매일 작업 내용 자동 정리
- 바이브코딩에 관심 있는 분들을 위한 강의자료로도 활용
- 영상 스크립트 자동 생성까지 확장 예정
